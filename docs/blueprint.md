# **App Name**: Portfolio Pro

## Core Features:

- Frontend Hosting: Host static portfolio website on Firebase Hosting, ensuring fast and secure content delivery.
- Asset Storage: Utilize Firebase Storage to store and serve high-resolution images and other large assets efficiently.
- Contact Form Submission: Handle contact form submissions via Firebase Functions, forwarding validated data to a specified email address.
- Analytics Integration: Use Firebase Analytics to track visitor behavior and understand which portfolio pieces are most engaging. No PII (personally identifiable information) will be collected.
- Automatic Image Optimization: Generative AI tool that automatically optimizes images upon upload to Firebase storage by reducing file sizes while maintaining quality, thus improving page load times.

## Style Guidelines:

- Primary color: candy red  to convey professionalism and stability.
- Background color: Very light gray for a clean, modern look.
- Accent color: Teal  for highlighting interactive elements and calls to action.
- Body font: 'Inter', a sans-serif font, will be used for a modern, readable design.
- Headline font: 'Space Grotesk', a sans-serif font, for impactful headlines.
- Use minimalist icons to represent skills and project categories.
- Employ a clean, grid-based layout to showcase portfolio pieces effectively.
- Implement subtle transition animations when navigating between portfolio items.
